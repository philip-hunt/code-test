# Code Test

1. [OPTIONAL] Create a virtual environment

   ```
    python3 -m venv venv
    source ./venv/bin/activate
   ```

1. Install dependencies

   ```
    pip3 install -r requirements.txt
   ```

1. Run the code

   ```
    python3 src/main.py
   ```

1. [OPTIONAL] Run the tests

   ```
    pytest
   ```
